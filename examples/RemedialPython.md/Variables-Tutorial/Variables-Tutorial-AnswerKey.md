### Quiz: Python Variables

1. What is the output of the following code?
   ```python
   x = 5
   print("x is", x)
   ```
   A) x is 5  
   B) 5  
   C) x is  
   D) Syntax error

2. Which of the following is a valid variable name in Python?
   A) my_variable  
   B) 123variable  
   C) my variable  
   D) _my_variable

3. What is the output of the following code?
   ```python
   a = 5
   b = 3
   a, b = b, a
   print(a, b)
   ```
   A) 5 3  
   B) 3 5  
   C) 3 3  
   D) 5 5
### Answer Key:

1. A) x is 5  
2. A) my_variable  
3. B) 3 5
